# Z Designs - Software Development Agency Website

A modern, professional website for Z Designs, a startup software development agency. Built with React, TypeScript, Tailwind CSS, and Motion (Framer Motion).

## Features

### Multi-Page Architecture
- **Home Page**: Hero section, services preview, benefits, tech stack, and CTA
- **Services Page**: Detailed service offerings with features
- **About Page**: Company story, mission, values, and team information
- **Projects Page**: Portfolio showcase with project cards
- **Contact Page**: Contact form with company information

### Design Elements
- **Dark Mode First**: Modern dark theme with zinc color palette
- **Gradient Accents**: Electric blue to violet gradients throughout
- **Smooth Animations**: Motion-powered transitions and scroll animations
- **Fully Responsive**: Mobile-first design that works on all devices
- **Modern Typography**: Inter font family with proper hierarchy

### UI Components
- **Navigation**: Sticky navigation with mobile menu and active state indicators
- **Footer**: Comprehensive footer with links and newsletter signup
- **Buttons**: Gradient primary buttons with hover effects
- **Cards**: Service cards, project cards, and feature cards
- **Forms**: Contact form with validation

### Tech Stack
- React 18.3.1
- TypeScript
- React Router DOM for routing
- Motion (Framer Motion) for animations
- Tailwind CSS for styling
- Lucide React for icons

## Pages Structure

### Home Page (`/`)
- Hero section with animated logo and CTA buttons
- Services preview with 4 main services
- "Why Choose Z Designs" benefits section
- Tech stack showcase
- Final CTA section

### Services Page (`/services`)
- Detailed service descriptions
- Feature lists for each service
- Alternating layout for visual interest

### About Page (`/about`)
- Company story and mission
- Statistics showcase
- Core values
- Team information

### Projects Page (`/projects`)
- Portfolio grid with 6 sample projects
- Project cards with gradients and tags
- Industry experience showcase

### Contact Page (`/contact`)
- Contact form (name, email, message)
- Contact information (email, phone, location)
- Social media links
- "What happens next" process overview

### Primary Colors
- Background: `zinc-950` (#0a0a0b)
- Secondary Background: `zinc-900` (#18181b)
- Text: `white` / `zinc-400`

### Accent Colors
- Blue: `#3b82f6` (blue-500)
- Violet: `#8b5cf6` (violet-500)
- Gradients: `from-blue-500 to-violet-500`

## Responsive Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## Key Features

1. **Smooth Scroll**: Implemented throughout the site
2. **Animation on Scroll**: Elements animate as they enter viewport
3. **Hover Effects**: Interactive hover states on buttons and cards
4. **Active Navigation**: Current page highlighted in navigation
5. **Mobile Menu**: Collapsible mobile navigation
6. **Form Validation**: Contact form with HTML5 validation
